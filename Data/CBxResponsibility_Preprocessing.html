Hypotheses

Hypothesis 1: If envy is a function of inequity, then the inequity manipulation will have an effect on episodic envy, and the blame manipulation will not.

Hypothesis 2: If anger is a function of blame, then the blame manipulation will have an effect on episodic anger, and the inequity manipulation will not.

Hypothesis 3: If money burning is a function of inequity aversion, then the inequity manipulation will have an effect on money burning, and the blame manipulation will not.

df <- df %>%
  mutate(
    exp_condition = factor(exp_condition)
  )

#create the contrasts 
C1_raw <- c( -1, 1, 1, -1 ) 
C2_raw <- c( -1, -1, 3, -1 )


n <- as.numeric(table(df$exp_condition))

# Convert contrasts to column vectors
C1 <- matrix(C1_raw, ncol = 1)
C2 <- matrix(C2_raw, ncol = 1)

# Weighted Gram-Schmidt: make C2 orthogonal to C1
proj <- sum(n * C1_raw * C2_raw) / sum(n * C1_raw^2)
C2_orth <- C2_raw - proj * C1_raw

# Numeric codes for factor levels
level_codes <- as.numeric(df$exp_condition)

# Assign contrast values to each row
df$C1 <- C1_raw[level_codes]      
df$C2 <- C2_orth[level_codes]

#checking the orthogonality 
cor.test(df$C1, df$C2)
